
# 강남구 상권 데이터 추출 및 필터링 실행 가이드 (CSV 전용)

사용자의 요청에 따라 Shapefile 및 시각화 과정을 제외하고, 제공해주신 `서울시 주요 82장소 목록(강남구).csv`를 기준으로 CSV 데이터를 추출하도록 프로세스를 간소화하였습니다.

## 1. 실행 환경 준비 (라이브러리 설치)

터미널(CMD 또는 PowerShell)을 열고 `d:\git_rk` 경로에서 아래 명령어를 실행하여 필요한 라이브러리를 설치해 주세요. (이미 설치되어 있다면 건너뛰셔도 됩니다.)

```bash
pip install pandas openpyxl
```

## 2. 데이터 추출 스크립트 실행

새롭게 작성된 목록 기반 추출 스크립트를 실행합니다. 이 스크립트는 `서울시 주요 82장소 목록(강남구).csv`의 `AREA_NM`을 기준으로 상권 코드를 매칭하여 10개의 CSV 파일을 필터링합니다.

```bash
python process_gangnam_data_v2.py
```

## 3. 결과물 확인

스크립트 실행이 완료되면 `d:\git_rk\REPORT\Gangnam_CSV_{timestamp}` 폴더가 생성되며, 그 내부에 필터링된 결과물이 저장됩니다.

- `gangnam_*.csv`: 강남구 목록과 매칭된 상권 데이터 (UTF-8 with BOM 형식, 엑셀 바로 보기 가능)

## 4. 참고 사항
- 이번 버전은 Shapefile 및 지도 시각화가 제외된 CSV 데이터 전용 추출 가이드입니다.
- 명칭 매칭이 되지 않는 경우를 대비하여 `자치구_코드_명` 기준 필터링 로직이 보조적으로 포함되어 있습니다.
