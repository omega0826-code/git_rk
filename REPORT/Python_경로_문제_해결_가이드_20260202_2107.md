# Antigravity 터미널에서 Python 경로 문제 해결 방법

## ✅ 현재 상태
**지도 생성 성공!** `d:\git_rk\output\서울시_82장소_지도.html` (580KB)

하지만 Antigravity 터미널에서 Python 명령이 실행되지 않는 문제가 있습니다.

---

## 🔍 Python 경로 확인 방법

### 1. Python 설치 위치 찾기

명령 프롬프트에서 실행:
```bash
where python
```

일반적인 Python 설치 경로:
- `C:\Python39\python.exe`
- `C:\Python310\python.exe`
- `C:\Users\user\AppData\Local\Programs\Python\Python39\python.exe`
- `C:\ProgramData\Anaconda3\python.exe`

### 2. 가상 환경 확인

프로젝트에 가상 환경이 있는지 확인:
```bash
dir d:\git_rk\.venv
```

가상 환경의 Python 경로:
- `d:\git_rk\.venv\Scripts\python.exe`

## 🔧 해결 방법

### 방법 1: 전체 경로로 Python 실행 (추천 ⭐)

Antigravity에서 `python` 대신 **전체 경로**를 사용:

```bash
# 예시 (실제 경로로 변경 필요)
C:\Python310\python.exe script.py

# 또는 가상 환경 사용
d:\git_rk\.venv\Scripts\python.exe script.py
```

#### Antigravity에서 사용할 스크립트 수정

`map_visualization.py`를 실행하는 배치 파일을 다음과 같이 수정:

```batch
@echo off
REM Python 전체 경로 지정
set PYTHON_PATH=C:\Python310\python.exe

REM 또는 가상 환경 사용
REM set PYTHON_PATH=d:\git_rk\.venv\Scripts\python.exe

%PYTHON_PATH% map_visualization.py > map_log.txt 2>&1
```

---

### 방법 2: 가상 환경 활성화 후 실행

가상 환경이 있다면 먼저 활성화:

```bash
# 가상 환경 활성화
d:\git_rk\.venv\Scripts\activate

# 그 다음 Python 실행
python map_visualization.py
```

Antigravity에서는 다음과 같이 실행:

```batch
@echo off
call d:\git_rk\.venv\Scripts\activate.bat
python map_visualization.py > map_log.txt 2>&1
```

---

### 방법 3: PATH 환경 변수에 Python 추가

**주의: 시스템 설정 변경이 필요합니다**

#### Windows 10/11에서 PATH 설정:

1. **시스템 환경 변수 열기**
   - Windows 키 + R
   - `sysdm.cpl` 입력 후 Enter
   - "고급" 탭 → "환경 변수" 클릭

2. **사용자 변수 또는 시스템 변수에서 Path 찾기**
   - "Path" 선택 → "편집" 클릭

3. **Python 경로 추가**
   - "새로 만들기" 클릭
   - Python 설치 경로 추가 (예: `C:\Python310`)
   - Python Scripts 경로도 추가 (예: `C:\Python310\Scripts`)

4. **확인 후 재시작**
   - 모든 창에서 "확인" 클릭
   - 명령 프롬프트를 **새로 열어서** 테스트

#### 확인:
```bash
python --version
```

---

### 방법 4: 배치 파일에서 직접 경로 설정 (임시 해결)

매번 전체 경로를 입력하기 번거롭다면, 배치 파일을 만들어 사용:

**`run_python.bat`**
```batch
@echo off
REM Python 경로를 임시로 PATH에 추가
set PATH=C:\Python310;C:\Python310\Scripts;%PATH%

REM 이제 python 명령 사용 가능
python %*
```

사용 방법:
```bash
run_python.bat map_visualization.py
```

## 🎯 Antigravity에서 사용할 최종 스크립트

Python 전체 경로를 찾은 후, 다음 배치 파일을 사용하세요:

**`run_map_fixed.bat`**
```batch
@echo off
chcp 65001 > nul

REM ========================================
REM Python 경로 설정 (실제 경로로 수정 필요)
REM ========================================

REM 옵션 1: 시스템 Python
set PYTHON=C:\Python310\python.exe

REM 옵션 2: 가상 환경 Python (주석 해제하여 사용)
REM set PYTHON=d:\git_rk\.venv\Scripts\python.exe

REM ========================================

echo 지도 생성 시작...
echo.

%PYTHON% map_visualization.py > map_log.txt 2>&1

if %ERRORLEVEL% EQU 0 (
    echo 성공! 결과: d:\git_rk\output\서울시_82장소_지도.html
) else (
    echo 오류 발생! 로그 확인: map_log.txt
)

pause
```
