## 1. 현재 한계

| 오타     | 올바른 표기                     | 유형      |
| -------- | ------------------------------- | --------- |
| 됬다     | **됐다**                        | 활용 오류 |

| 한꺼번에 | 한꺼번에 (정상)                 | 띄어쓰기  |
| 때 문에  | **때문에**                      | 띄어쓰기  |

## 2. 구현 방안

### 2-1. py-hanspell (네이버 맞춤법 검사기 API)

**장점**: 가장 정확한 한국어 맞춤법 검사  
**단점**: 네트워크 필요, API 제한(500자), 비공식 API

```bash
pip install py-hanspell
```

```python
from hanspell import spell_checker

def check_spelling_hanspell(lines: List[str]) -> List[Issue]:
    issues = []
    for idx, line in enumerate(lines, 1):
        stripped = line.strip()
        if len(stripped) < 5:
            continue
        
        # 500자 단위로 분할 (API 제한)
        chunks = [stripped[i:i+500] for i in range(0, len(stripped), 500)]
        
        for chunk in chunks:
            try:
                result = spell_checker.check(chunk)
                if result.errors:
                    for error_word, suggestion in result.words.items():
                        if error_word != suggestion:
                            issues.append(Issue(
                                category='text',
                                severity='오타',
                                location=f'줄 {idx}',
                                original=error_word,
                                description=f'맞춤법 오류',
                                suggestion=suggestion,
                            ))
            except Exception as e:
                pass  # API 오류 시 건너뛰기
    
    return issues
```

**주의사항**:
- `spell_checker.check()`는 500자 제한 → 분할 전송 필요
- 네트워크 지연 → 대량 문단에서 느림 (rate limiting 고려)
- 비공식 API이므로 변경/차단 가능성 있음
- 결과에서 `words` dict의 key=원문, value=교정본 비교

## 3. 권장 구현 전략

### 단계별 적용

| 단계      | 도구            | 목적                   | 난이도  |
| --------- | --------------- | ---------------------- | ------- |
| **1단계** | **kiwi**        | 띄어쓰기 교정          | ⭐ 낮음  |
| **2단계** | **py-hanspell** | 맞춤법 검사 (온라인)   | ⭐⭐ 보통 |
| **3단계** | **hunspell**    | 맞춤법 검사 (오프라인) | ⭐⭐ 보통 |

### 통합 방식

```python
class SpellChecker:
    """맞춤법 + 띄어쓰기 검사 (C-2단계)"""
    
    def __init__(self, lines: List[str], mode: str = 'auto'):
        """
        mode:
          'auto'    - kiwi + hanspell (온라인 시) or kiwi + hunspell (오프라인 시)
          'online'  - kiwi + hanspell
          'offline' - kiwi + hunspell
          'kiwi'    - kiwi만 (띄어쓰기만)
        """
        self.lines = lines
        self.mode = mode
        self.issues: List[Issue] = []
    
    def check_all(self) -> List[Issue]:
        self._check_spacing_kiwi()
        
        if self.mode in ('auto', 'online'):
            try:
                self._check_spelling_hanspell()
            except Exception:
                if self.mode == 'auto':
                    self._check_spelling_hunspell()
        elif self.mode == 'offline':
            self._check_spelling_hunspell()
        
        return self.issues
```
