## 3. Excel 파일 변경 시 수정 포인트

### 데이터만 변경된 경우
→ **수정 불필요**, 스크립트 재실행만 하면 됨

### 표 추가/구조 변경 시
→ `TABLES` 리스트에 `Cfg()` 추가/수정 필요

```python
# 예: 19번째 표 추가
Cfg(19, "신규 표 제목", total_row=400, name_row=398, chart="hbar")
```

| 파라미터    | 설명                    | 확인 방법                                  |
| ----------- | ----------------------- | ------------------------------------------ |
| `total_row` | "전 체" 행 번호         | Excel에서 A열 검색                         |
| `name_row`  | 항목명 행 번호          | 항목명이 있는 행                           |
| `data_col`  | 데이터 시작 열 (기본 4) | 첫 빈도 값 열                              |
| `chart`     | 차트 유형               | `hbar`, `pie`, `donut`, `radar`, `stacked` |

## 4. PowerPoint에서 차트 편집

### 데이터 수정
1. 차트를 **더블클릭** → 차트 편집 모드
2. **데이터 편집** 클릭 → 내장 스프레드시트
3. 값 수정 후 외부 클릭 → 차트 자동 업데이트

### 디자인 수정
- 차트 선택 → **차트 디자인** 탭 → 색상/스타일 변경
- 데이터 라벨 더블클릭 → 폰트/크기/색상 변경
- 축 더블클릭 → 축 서식 변경

### 워드 프로세서 삽입
- 슬라이드에서 차트 선택 → `Ctrl+C`
- 워드에서 `Ctrl+V` → **그림 붙여넣기** 또는 **개체 붙여넣기**
- 흰 배경이므로 별도 처리 불필요
