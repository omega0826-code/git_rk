# 📊 차트 디자인 총괄 가이드라인 (Master)

> 이 문서는 **모든 테마에 공통 적용**되는 차트 디자인 규칙입니다.
> 테마별 색상/폰트 변경은 `themes/` 폴더의 개별 테마 파일을 참조하세요.

---

## 1. 차트 유형 선택 기준

| 차트 유형     | 용도               | 적합한 데이터       | 권장 항목 수 |
| ------------- | ------------------ | ------------------- | ------------ |
| **수평 막대** | 항목별 비율 비교   | 단일/중복 응답      | 5~12개       |
| **파이**      | 구성비             | 단일 응답 (합=100%) | 3~6개        |
| **도넛**      | 구성비 (중심 여백) | 이분형/소수 항목    | 2~5개        |
| **레이더**    | 다항목 균형 비교   | 평균 점수, 만족도   | 5~8개        |
| **누적 막대** | 그룹별 교차 비교   | 대학별·지역별       | 상위 10개    |

```
항목 ≤ 5 → 파이/도넛
항목 > 5 → 수평 막대
평균 점수 → 레이더
그룹 교차 → 누적 막대
```

## 2. 슬라이드 레이아웃 (고정)

| 속성               | 값                                     |
| ------------------ | -------------------------------------- |
| 슬라이드           | **13.333 × 7.5"** (16:9)               |
| 차트 영역 (막대)   | `left=0.5", top=1.0", w=12.3", h=5.9"` |
| 차트 영역 (파이)   | `left=1.5", top=1.2", w=10", h=5.5"`   |
| 차트 영역 (레이더) | `left=2.0", top=1.2", w=9", h=5.5"`    |

### 텍스트 요소 위치 (고정)

| 요소        | 위치             | 정렬 |
| ----------- | ---------------- | ---- |
| 제목        | `0.5", 0.3"`     | 좌측 |
| 부제 (n=)   | 제목 우측 인라인 | 좌측 |
| 단위        | `10.5", 0.35"`   | 우측 |
| 페이지 번호 | `12", 7.05"`     | 우측 |

## 3. 데이터 라벨 포맷 (고정)

| 차트      | 형식           | 예시          | 위치        |
| --------- | -------------- | ------------- | ----------- |
| 수평 막대 | `비율% (빈도)` | `50.5% (505)` | OUTSIDE_END |
| 파이/도넛 | `비율% (빈도)` | `52.1% (521)` | 파이 위     |
| 레이더    | `0.00`         | `3.75`        | 마커 근처   |
| 누적 막대 | `#,##0`        | `156`         | CENTER      |

### 규칙
- 빈도에 단위(`명`) 포함하지 않음
- 0% 항목 제외
- 수평 막대: 비율 오름차순 정렬
- 시리즈명: 공백 `' '` (차트 제목 방지)

## 4. 축 처리 (고정)

| 항목             | 방식                                  |
| ---------------- | ------------------------------------- |
| 카테고리 축 라벨 | 표시 (테마별 색상)                    |
| 카테고리 축 선   | 0.5pt (테마별 색상)                   |
| 값 축 전체       | **투명 처리** (배경색 동일 폰트, 2pt) |
| 보조선           | 모두 OFF                              |
| 축 제목          | 사용 안 함                            |

## 6. 테마 적용 방법

### 테마 파일 구조
```
scripts/docs/design/
├── chart_design_master.md       ← 이 문서 (공통 규칙)
└── themes/
    ├── white_clean.md           ← 현재 적용 중
    ├── dark_premium.md          ← 진남색 배경 테마
    └── (추가 테마)
```

### 테마 파일에 포함되는 항목 (변경 가능)
| 항목        | 설명                      |
| ----------- | ------------------------- |
| 배경색      | 슬라이드 배경             |
| 텍스트 색상 | 제목, 라벨, 축, 보조      |
| 차트 팔레트 | 10색                      |
| 장식 요소   | 악센트 바, 테두리 등 유무 |
| 폰트 (선택) | 특수 폰트 지정 시         |

### 코드 적용
스크립트의 `Theme` 클래스만 수정하면 테마 전환 가능:
```python
@dataclass
class Theme:
    # ← 테마 파일의 값으로 변경
    bg: Tuple = (0xFF, 0xFF, 0xFF)
    black: Tuple = (0x22, 0x22, 0x22)
    ...
```
