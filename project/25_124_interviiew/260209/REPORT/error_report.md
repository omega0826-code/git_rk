## 1. 데이터 구조 관련

### 1-1. 빈 행 존재 (22행)
- 원본 CSV에 **22행**이 존재하나, 마지막 22행은 빈 데이터임
- 실제 유효 기업 수: **21개** (정상)

### 1-2. 컬럼명 불일치
- 원본 CSV 컬럼명: `순번`, `업체명`, `근로자수(명)`, `규모`, `업종`, `업종 상세`, `사업체 유형`, `Q1`~`Q8`, `Q_ETC`
- `column.md` 파일의 `NAME` 컬럼은 CSV의 `업체명`에 해당하며, `NAME`이라는 컬럼은 CSV에 존재하지 않음

## 3. 규모 분류 특이사항

### 3-1. 맑은기업 — 근로자 20명, 규모 재분류
- CSV 원본: `근로자수(명)=20`, `규모=30~50명 미만`
- 근로자 수(20명) 기준으로 "30명 미만"에 해당하므로, CSV의 사전 분류를 수정하여 **30명 미만**으로 재분류
- **최종 조치**: 30명 미만 = **8개**, 30~50명 미만 = **8개**로 변경 (기존: 7/9)

## 5. 업종 분류 관련

### 5-1. 대송컨테이너항업 — "유통/소비/서비스업"
- CSV 원본: `업종=유통/소비/서비스업`, `업종 상세=화물운송업`
- 기존 보고서에서 "서비스업"으로 분류 → **정확**
- 보고서의 업종 분류(서비스업 1개)와 일치함

### 5-2. 코스포영남파워, 드래곤엠앤이, 디비밸리 — "그 외 기타업"
- 3개 기업 모두 CSV에서 `업종=그 외 기타업`으로 분류
- 보고서의 "기타업 3개"와 일치 → **정상**

### 5-3. ㈜성전사 — "건설업"
- CSV 원본: `업종=건설업`, `업종 상세=금속기계`
- 업종 상세가 "금속기계"이나 건설업으로 분류됨
- 기존 보고서에서 건설업 2개(세영윈도우, ㈜성전사)로 분류 → **원본 CSV 기준 정확**

## 8. 요약

- **데이터 정합성**: 기업 수, 업종/규모 분류, 사업체 유형 등 **모두 일치** 확인
- **주의 필요 항목**: 맑은기업 근로자수(20명) ↔ 규모분류(30~50명) 불일치는 원본 CSV의 사전 분류를 그대로 사용
- **표기 통일**: 기업명 띄어쓰기 및 ㈜ 표기는 보고서 내 통일 적용
- **응답 누락**: 6개 기업에서 일부 항목 미응답이 있으나, 보고서 작성에 영향을 미치지 않는 수준
