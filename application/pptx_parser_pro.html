사용 가이드

PPTX Parser Pro는 PowerPoint 파일을 브라우저에서 분석하는 도구입니다. 서버 업로드 없이 로컬에서만 처리됩니다.

1단계 — PPTX 파일을 드래그하거나 클릭하여 업로드
2단계 — 좌측에서 슬라이드를 클릭하여 상세 확인
3단계 — 우측 패널에서 파싱 데이터 / 디자인 템플릿 확인
4단계 — 상단 "내보내기" 버튼으로 JSON / CSV / ZIP 다운로드
분석 항목:
텍스트 (폰트, 크기, 색상, 위치) · 표 (행/열, 병합, 스타일) · 차트 (유형, 데이터) · 이미지 (추출, 위치) · 도형 · 슬라이드 레이아웃 유형 · 색상 팔레트 · 폰트 목록
지원 형식: .pptx (PowerPoint 2007+)
로컬 실행: 이 HTML 파일을 브라우저에서 열기만 하면 됩니다.
