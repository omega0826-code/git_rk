# 의원급 피부과 입지분석 EDA 실행 계획

의원급 피부과 최적 입지 선정을 위한 가이드라인 v3.01에 따라 데이터 분석을 수행합니다.

## 분석 개요
- **대상**: 강남구 주요 5개 상권 (양재역, 가로수길, 강남역, 역삼역, 선릉역)
- **목적**: 의원급 피부과 최적 입지 선정을 위한 EDA 수행
- **데이터**: 2022~2024년 상권 분석 데이터

## 단계별 분석 계획

### 1. Phase 0: 데이터 로딩 및 초기 설정
가이드라인의 개선된 `01_데이터로딩.py` 스크립트를 실행하여 분석 환경을 조성하고 데이터를 로드합니다.

### 2. Phase 1~5: 단계별 EDA 실행
각 주제별로 Python 스크립트를 작성하여 실행합니다.
- 경쟁 환경 분석 (시장 규모, 개폐업률)
- 고객 분석 (타겟 고객 매출, 방문 패턴)
- 인구 및 유동 분석 (직장/상주 인구, 유동인구)
- 입지 조건 분석 (접근성, 의료 인프라, 소비력)
- 종합 평가 (통합 스코어링 및 레이더 차트)

### 3. Phase 6: 결과 종합 및 리포팅
모든 분석 결과를 종합하여 마크다운 형식의 최종 보고서를 생성합니다.

## 결과물 저장 구조
- `REPORT/01_경쟁환경분석/` ~ `REPORT/05_종합평가/`: 각 단계별 CSV 및 시각화 결과
- `REPORT/06_최종리포트/`: 종합 분석 보고서
- `REPORT/00_스크립트/`: 분석에 사용된 전체 코드
