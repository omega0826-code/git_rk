# 병원정보조회 웹 애플리케이션 - 로컬 실행 가이드

## 🚀 빠른 시작 (3단계)

### 1단계: 실행.bat 더블 클릭
```
실행.bat 파일을 더블 클릭하세요
```

### 2단계: 브라우저 자동 실행
- 웹 서버가 자동으로 시작됩니다
- 브라우저가 자동으로 열립니다
- `http://localhost:8080/병원정보조회_웹앱.html` 주소로 접속됩니다

### 3단계: API 인증키 입력 후 사용
- 공공데이터포털에서 발급받은 **디코딩 인증키**를 입력하세요
- 검색 조건을 선택하고 검색하세요

## 📋 필수 요구사항

### Python 설치 필요
이 웹 애플리케이션은 Python 내장 웹 서버를 사용합니다.

**Python이 설치되어 있지 않은 경우:**
1. [Python 공식 사이트](https://www.python.org/downloads/) 방문
2. Python 3.x 최신 버전 다운로드
3. 설치 시 **"Add Python to PATH"** 체크 필수
4. 설치 완료 후 `실행.bat` 다시 실행

**Python 설치 확인:**
```bash
python --version
```

## ❓ 문제 해결

### "Python이 설치되어 있지 않습니다" 오류
**해결방법:**
1. Python 설치 (위의 필수 요구사항 참조)
2. 설치 후 컴퓨터 재시작
3. `실행.bat` 다시 실행

### "포트 8080이 이미 사용 중입니다" 오류
**해결방법:**
1. 다른 프로그램이 8080 포트를 사용 중입니다
2. `서버실행.py` 파일을 열어 `PORT = 8080`을 `PORT = 8081`로 변경
3. 다시 실행

### 브라우저가 자동으로 열리지 않음
**해결방법:**
1. 수동으로 브라우저 열기
2. 주소창에 `http://localhost:8080/병원정보조회_웹앱.html` 입력

### API 호출이 안 됨
**해결방법:**
1. 인터넷 연결 확인
2. **디코딩 인증키**를 사용하고 있는지 확인
3. 인증키 발급 후 30분 이상 경과했는지 확인

### 왜 로컬 서버가 필요한가요?
- 브라우저의 CORS(Cross-Origin Resource Sharing) 보안 정책 때문입니다

- 로컬 웹 서버(`http://` 프로토콜)를 통하면 정상적으로 작동합니다

## 🔑 API 인증키 사용 가이드

### 디코딩 인증키 사용 필수
- ✅ **올바름**: `abc123+def456==` (디코딩 인증키)
- ❌ **잘못됨**: `abc123%2Bdef456%3D%3D` (인코딩 인증키)

### 인증키 확인 방법
1. [공공데이터포털](https://www.data.go.kr) 로그인
2. [마이페이지 > 인증키 발급현황](https://www.data.go.kr/iim/api/selectApiKeyList.do)
3. 병원정보서비스 API 찾기
4. **"일반 인증키(Decoding)"** 열의 값 복사
