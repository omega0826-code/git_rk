# 요청사항
 - openapi를 이용한 병원정보조회 및 다운로드 웹 애플리케이션 개발
 - openapi 활용 가이드 라인을 참고하여 개발
 - 참고 웹 애플리케이션 구조를 참고하여 개발

# 참고파일 위치
 - 활용 가이드 라인 : D:\git_rk\openapi\getHospBasisList\getHospBasisList_api_guideline.md
 - 참고 웹 애플리케이션 : D:\git_rk\openapi\sample\사업자등록 상태조회(오픈API 활용, 엑셀용).html

# 산출물
 - 모든 산출물은 폴더 내에 포함
 - 개발과 테스트를 위한 모든 파일이 포함
 - 개발에 검토한 내용은 마크다운 형태로 모두 저장
 - 추후 다른 개발자들이 개발 내용을 이해할 수 있도록 주석을 추가
 - 다른 openapi를 이용한 애플리케이션 개발에 활용할 수 있도록 가이드라인을 제작
