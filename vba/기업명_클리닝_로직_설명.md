# 기업명 클리닝 로직 설명

## 목적
기업명 중복 검사를 위해 선택된 기업명들을 표준화된 형식으로 변환

## 처리 단계

### 1. 영문 → 한글 변환
- 영어로 된 기업명을 한글로 변환
- 예: "Samsung" → "삼성"

### 2. 공백 제거
- 모든 띄어쓰기 삭제
- 예: "삼성 전자" → "삼성전자"

### 3. 법인 형태 표기 제거
다음과 같은 법인 형태 표기를 삭제:
- 주식회사
- (주)
- ㈜
- 유한회사
- (유)
- ㈜
- 재단법인
- 사단법인
- 합자회사
- 합명회사
- 유한책임회사
- Co., Ltd.
- Corporation
- Inc.
- LLC

### 4. 불필요한 부가 정보 제거
- 괄호 안의 내용 삭제: "(본사)", "(공장)", "(지점)" 등
- 공장, 지점, 본사 등의 단어 삭제
- 특수문자 제거: -, _, ., 등

### 5. 최종 정리
- 남은 공백 제거
- 대소문자 통일 (영문이 남아있는 경우)

## 예시

| 원본 기업명 | 클리닝 후 |
|------------|----------|
| 주식회사 가나 | 가나 |
| 가나 주식회사 | 가나 |
| (주)가나 | 가나 |
| ㈜가나다라(본사) | 가나다라 |
| Samsung Electronics Co., Ltd. | 삼성전자 |
| 삼성 전자 주식회사 | 삼성전자 |
| 현대자동차(주) 울산공장 | 현대자동차 |

## 중복 판정
클리닝 후 동일한 문자열이 되면 같은 기업으로 판정
