### ⚙️ 스크립트 사양

| 항목        | 값                         |
| ----------- | -------------------------- |
| Python 버전 | 3.14+                      |
| 필수 패키지 | `pandas`, `openpyxl`       |
| 인코딩      | UTF-8 BOM (`utf-8-sig`)    |
| 실행 위치   | `data/factory_on/scripts/` |

### 사용법
```bash
python convert_xlsx_to_csv.py
python convert_xlsx_to_csv.py --raw-dir /path/to/raw
```
