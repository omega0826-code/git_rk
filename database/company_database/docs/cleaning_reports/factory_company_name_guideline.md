# 공장기업명 정제 가이드라인

> 전국공장등록현황 데이터의 회사명 정규화 및 중복 검출 표준 절차
> 기존 `company_name_dedup_guideline.md`를 그대로 적용

---

## 1. 컬럼 구조

| 컬럼명            | 설명                                 |
| ----------------- | ------------------------------------ |
| **회사명**        | 원본 회사명 (수정 불가, 이력 보존용) |
| **회사명_정규화** | 정규화된 회사명 (중복 비교에 사용)   |
| **회사명_영문**   | 회사명에 포함된 영문 부분만 추출     |
| **비고**          | `[중복]`, `[영문포함]` 등 태그       |

## 2. 정규화 규칙

기존 `company_name_dedup_guideline.md`의 규칙을 동일하게 적용:
1. 법인 표기 제거: `(주)`, `(유)`, `주식회사`, `유한회사`, `농업회사법인`, `영농조합법인`, `사회적협동조합`, `㈜` 등
2. 괄호 및 괄호 내용 제거
3. 공백/특수문자 제거
4. 영어 소문자 통일

## 3. 중복 판정
- `회사명_정규화` 값이 동일한 행이 2건 이상이면 중복으로 판정
- 비고란에 `[중복] N건 - nid1, nid2, ...` 형식으로 기록

## 4. 참고 사항
- 원본 `회사명` 컬럼은 절대 수정하지 않음
- 중복은 정보 제공 목적이며, 삭제 대상은 아님
