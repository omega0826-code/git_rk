# 기업명 중복검사 가이드라인

> 참여기업 리스트의 기업명 정규화 및 중복 검출을 위한 표준 절차

---

## 1. 컬럼 구조

| 컬럼명              | 설명                                                    |
| ------------------- | ------------------------------------------------------- |
| **기업명_원본**     | 원본 기업명 (수정 불가, 이력 보존용)                    |
| **기업명_중복검사** | 정규화된 기업명 (중복 비교에 사용)                      |
| **기업명_영문**     | 기업명에 포함된 영문 부분만 추출                        |
| **비고**            | 특이사항 기록 (`[중복]`, `[영문포함]`, `[주소수정]` 등) |

---

## 2. 기업명 정규화 규칙 (순서대로 적용)

### 2-1. 법인 표기 제거

| 제거 대상                                                  |
| ---------------------------------------------------------- |
| `(주)`, `(유)`, `(한)`, `(사)`                             |
| `주식회사`, `유한회사`, `유한책임회사`                     |
| `농업회사법인`, `영농조합법인`, `사회적협동조합`           |
| `사단법인`, `재단법인`, `협동조합`, `합자회사`, `합명회사` |

### 2-2. 괄호 및 괄호 내용 제거
```
에본(주) (EVON Co. Ltd)  →  에본
(주)엠에프에스티(MFST)   →  엠에프에스티
```

### 2-3. 공백 및 특수문자 제거
- 모든 공백(띄어쓰기) 제거
- `-`, `_`, `.` 등 특수문자 제거

### 2-4. 영어 소문자 통일
```
BIOBIO → biobio
MK2바이오 → mk2바이오
```

### 정규화 예시

| 기업명_원본                | 기업명_중복검사 | 기업명_영문 |
| -------------------------- | --------------- | ----------- |
| (유)대한물산               | 대한물산        |             |
| (주)BIOBIO                 | biobio          | BIOBIO      |
| 농업회사법인(주)건강한약초 | 건강한약초      |             |
| 에본(주) (EVON Co. Ltd)    | 에본            | EVON Co Ltd |
| DHU메디코스                | dhu메디코스     | DHU         |

---

## 3. 영문 추출 규칙

- 기업명에서 영어 알파벳(`A-Za-z`)으로 구성된 단어를 모두 추출
- 추출된 영문은 원본 형태(대소문자 유지)로 `기업명_영문` 컬럼에 저장
- 영문이 없으면 공란
- 비고란에 `[영문포함] 추출된영문` 자동 기록

---

## 4. 중복 판정 규칙

- `기업명_중복검사` 값이 **동일**한 행이 2건 이상이면 중복으로 판정
- 비고란에 `[중복] N건 - Row X, Y, Z` 형식으로 기록
- 중복 판단 시 **영문은 소문자 통일 후 비교** (한글 변환하지 않음)

---

## 5. 비고란 자동 기록 태그

| 태그           | 의미                            | 예시                            |
| -------------- | ------------------------------- | ------------------------------- |
| `[중복]`       | 기업명 정규화 후 동일 기업 존재 | `[중복] 2건 - Row 5, 120`       |
| `[영문포함]`   | 기업명에 영어 포함              | `[영문포함] BIOBIO`             |
| `[주소수정]`   | 주소 클리닝 시 수정             | `[주소수정] 경산북도->경상북도` |
| `[주소불완전]` | 상세주소 누락                   | `[주소불완전] 시군구 이하 누락` |

---

## 6. 향후 확장 포인트

1. **영문 한글 음차 매핑 테이블**: 반복 등장하는 영문 약어에 대한 수동 매핑 테이블 구축
2. **사업자등록번호 기반 중복검사**: 기업명 외 사업자번호를 활용한 정밀 중복 판별
3. **유사 기업명 검출**: 편집 거리(Levenshtein Distance) 기반 유사 기업명 후보 추출
