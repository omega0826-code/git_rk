# 전화번호 정제 가이드라인

> 전국공장등록현황 데이터의 전화번호 컬럼 표준 전처리 절차

---

## 1. 허용 범위

| 유형             | 패턴              | 예시                           |
| ---------------- | ----------------- | ------------------------------ |
| 서울 지역번호    | `02-XXX(X)-XXXX`  | `02-585-5152`, `02-2088-2888`  |
| 지역번호 (3자리) | `0XX-XXX(X)-XXXX` | `031-531-6863`, `054-7487-708` |
| 인터넷전화       | `070-XXXX-XXXX`   | `070-7723-9277`                |
| 휴대폰           | `010-XXXX-XXXX`   | `010-1234-5678`                |

## 2. 비허용 (삭제 대상)

| 유형      | 예시                      | 사유             |
| --------- | ------------------------- | ---------------- |
| 빈칸      | ` `                       | 전화번호 없음    |
| ARS 번호  | `ARS-1833-2334`           | 특수번호         |
| 대표번호  | `18001435`, `1600-1064`   | 특수번호(1XXX)   |
| 무효 패턴 | `054-7487-708` (끝 3자리) | 유효 패턴 불일치 |

## 3. 정제 규칙

1. **공백/특수문자 정규화**: 괄호, 점, 공백을 하이픈으로 통일
2. **자동 포맷팅**: 하이픈 없는 숫자열을 지역번호/070/010 패턴으로 변환 시도
3. **유효성 검증**: 정규식 패턴 매칭으로 최종 확인

## 4. 삭제 기준
- 주소/전화번호 중 **1개라도 결함이면** 삭제 데이터로 이동
