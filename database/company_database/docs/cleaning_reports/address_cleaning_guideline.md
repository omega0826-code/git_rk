# 사업장 소재지 주소 클리닝 가이드라인

> 대구한의대학교 참여기업 리스트 등 주소 데이터를 포함한 Excel 파일의 표준 전처리 절차

---

## 1. 컬럼 구조

| 컬럼명                  | 설명                                |
| ----------------------- | ----------------------------------- |
| **시도**                | 주소에서 추출한 광역시/도 (정제 후) |
| **시군구**              | 주소에서 추출한 시/군/구 (정제 후)  |
| **사업장소재지_원본**   | 원본 주소 (수정 불가, 이력 보존용)  |
| **사업장소재지_클리닝** | 전처리 완료된 주소                  |

---

## 2. 전처리 규칙 (순서대로 적용)

### 2-1. 공백 정규화
- 앞뒤 공백 제거 (`strip`)
- 연속 공백을 단일 공백으로 치환 (`re.sub(r'\s+', ' ', addr)`)

### 2-2. 시도 오타 수정

| 오타         | 정정       |
| ------------ | ---------- |
| 경산북도     | 경상북도   |
| 대구시       | 대구광역시 |
| 대구광역시시 | 대구광역시 |
| 대구광여깃   | 대구광역시 |
| 부삭광역시   | 부산광역시 |

> [!TIP]
> 새로운 오타가 발견되면 이 테이블에 추가하고 스크립트의 `SIDO_CORRECTIONS` 딕셔너리에도 반영합니다.

### 2-3. 시도 약어 확장

| 약어 | 정식 명칭      |
| ---- | -------------- |
| 서울 | 서울특별시     |
| 부산 | 부산광역시     |
| 대구 | 대구광역시     |
| 인천 | 인천광역시     |
| 광주 | 광주광역시     |
| 대전 | 대전광역시     |
| 울산 | 울산광역시     |
| 세종 | 세종특별자치시 |
| 경기 | 경기도         |
| 강원 | 강원특별자치도 |
| 충북 | 충청북도       |
| 충남 | 충청남도       |
| 전북 | 전북특별자치도 |
| 전남 | 전라남도       |
| 경북 | 경상북도       |
| 경남 | 경상남도       |
| 제주 | 제주특별자치도 |

### 2-4. 시군구 기반 시도 추론
주소 첫 단어가 시도가 아니라 시군구인 경우, 해당 시군구가 속한 시도를 앞에 삽입합니다.

| 시군구 | 추론 시도 |
| ------ | --------- |
| 경산시 | 경상북도  |

> [!IMPORTANT]
> 시군구 → 시도 매핑은 프로젝트 특성에 따라 확장이 필요합니다. 동일 시군구명이 여러 시도에 존재할 수 있으므로 주의가 필요합니다.

## 3. 시도 / 시군구 추출 방법

### 추출 로직
```python
# 전처리 완료된 주소에서 공백으로 분리
parts = cleaned_addr.split()
sido    = parts[0] if len(parts) >= 1 else ''  # 첫 번째 토큰
sigungu = parts[1] if len(parts) >= 2 else ''  # 두 번째 토큰
```

### 유효한 시도 목록 (17개)
```python
VALID_SIDO = {
    '서울특별시', '부산광역시', '대구광역시', '인천광역시', '광주광역시',
    '대전광역시', '울산광역시', '세종특별자치시', '경기도', '강원도',
    '강원특별자치도', '충청북도', '충청남도', '전라북도', '전북특별자치도',
    '전라남도', '경상북도', '경상남도', '제주특별자치도'
}
```

### 검증
추출된 시도 값이 `VALID_SIDO`에 포함되지 않으면 수동 확인 대상으로 분류합니다.

## 4. 처리 결과 요약 (대구한의대 참여기업 리스트)

## 6. 향후 확장 포인트

1. **도로명/지번 분리**: 시군구 이후 주소를 읍면동, 도로명, 번지 등으로 추가 분리
2. **우편번호 매핑**: 클리닝된 주소 기반으로 우편번호 자동 매핑
3. **좌표 변환**: 지오코딩을 통한 위경도 추출
4. **시군구 → 시도 매핑 확장**: 전국 시군구 리스트 기반 자동 추론 로직 강화
5. **정규표현식 기반 패턴 매칭**: `~구`, `~시`, `~군`, `~도` 접미사를 활용한 자동 분류
